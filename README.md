Main features:

  * Micro-controler: ATMEGA328
  * Main voltage: 3.7 V
  * Radio card: nRF24L01
  * Sensor: Temperature DS18B20
  * Display: OLED
  * Buttons: 4
  * Setup output: 5V
  * Power controls: 3 MOSFET 2N7000
  * Analog reader: 1 pin at J9
  * Free pins: 1
  * Programming: Onboard serial programming
